# MindPulse – Frontend (Dashboard & Visualization)

## Overview
This repository contains the frontend of MindPulse, providing a real-time interface for emotion detection and visualization.

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- Chart.js  
- Three.js  

## Features
- Real-time emotion display  
- Interactive dashboard  
- Graph-based mood analysis  
- Dynamic UI elements  

## Workflow
1. Captures user input (camera/audio)  
2. Sends data to backend API  
3. Receives prediction results  
4. Displays emotions and updates graphs  

## Visual Features
- Emotion trend graphs using Chart.js  
- Interactive background using Three.js  

## Note
Backend API is implemented separately and handles model inference.
